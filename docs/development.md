# Development

## Workflow
1. Create a focused branch.
2. Ask Codex to implement one small task.
3. Run `pytest`, `ruff`, and `mypy` locally.
4. Review the git diff.
5. Commit with a conventional commit message.
6. Push the branch.
7. Open a GitHub pull request.
8. Merge with a merge commit.
9. Delete the branch.
10. Pull `main` locally.
